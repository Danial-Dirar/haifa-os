# Haifa OS

Arch-based distribution focused on a smooth out-of-the-box experience for:
- Gamers
- Developers
- Apple device users (leavers?)

Repo layout:
- `iso/`      ISO build (archiso profile)
- `packages/` Haifa OS packages (branding, defaults, meta packages)
- `artwork/`  Logos, wallpapers, themes
- `repo/`     Pacman repo output (generated)
