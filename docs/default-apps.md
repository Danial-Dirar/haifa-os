# Haifa OS — Default Applications

These are the applications that ship pre-installed on Haifa OS.

## Desktop Environment

| Role | Application | Package |
|------|-------------|---------|
| Desktop | KDE Plasma | `plasma-desktop` |
| Display manager | SDDM | `sddm` |
| Terminal | Konsole | `konsole` |
| File manager | Dolphin | `dolphin` |
| Text editor | Kate | `kate` |
| PDF viewer | Okular | `okular` |
| Image viewer | Gwenview | `gwenview` |
| Archive manager | Ark | `ark` |
| Screenshot | Spectacle | `spectacle` |
| System monitor | KSysGuard | `plasma-systemmonitor` |

## Web & Communication

| Role | Application | Package |
|------|-------------|---------|
| Browser | Firefox | `firefox` |

## Gaming

| Role | Application | Package |
|------|-------------|---------|
| Game store / launcher | Steam | `steam` |
| Non-Steam game launcher | Lutris | `lutris` |
| Game performance boost | Gamemode | `gamemode` |
| In-game overlay | MangoHud | `mangohud` |
| Gamescope (nested compositor) | Gamescope | `gamescope` |
| Windows games compatibility | Wine | `wine` |
| Wine helper | Winetricks | `winetricks` |

## System Tools

| Role | Application | Package |
|------|-------------|---------|
| Package manager (GUI) | Discover | `discover` |
| AUR helper | paru | `paru` (post-install) |
| Bluetooth | Bluedevil | `bluedevil` |
| Disk usage | Filelight | `filelight` |

## Audio

| Role | Application | Package |
|------|-------------|---------|
| Audio server | PipeWire | `pipewire` |
| PulseAudio compat | PipeWire-Pulse | `pipewire-pulse` |
| Session manager | WirePlumber | `wireplumber` |

## Installer

| Role | Application | Package |
|------|-------------|---------|
| Graphical installer | Calamares | `calamares` |

## Optional DEs (available at install time via Calamares)

- KDE Plasma (default)
- GNOME
- Hyprland
